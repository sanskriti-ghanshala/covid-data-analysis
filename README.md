# COVID-19 Data Analysis Project

## Overview

This project focuses on analyzing the global COVID-19 dataset using Python. The aim of this analysis is to explore the number of confirmed cases, deaths, recoveries, and active cases across different countries and WHO regions. By performing exploratory data analysis (EDA), we identify patterns, correlations, and trends in the data.

The project demonstrates basic data science skills such as data cleaning, data visualization, and statistical analysis using popular Python libraries.

## Dataset

The dataset contains COVID-19 statistics for multiple countries with the following columns:

* Country/Region
* Confirmed Cases
* Deaths
* Recovered
* Active Cases
* New Cases
* New Deaths
* New Recovered
* Deaths per 100 Cases
* Recovered per 100 Cases
* Deaths per 100 Recovered
* Confirmed Last Week
* 1 Week Change
* 1 Week % Increase
* WHO Region

## Tools and Libraries Used

The following Python libraries were used for data analysis and visualization:

* Pandas – Data manipulation and analysis
* NumPy – Numerical operations
* Matplotlib – Data visualization
* Seaborn – Statistical data visualization

## Project Workflow

1. Data Loading
   The dataset is loaded using the Pandas library.

2. Data Exploration
   Basic dataset information is explored using functions like `head()`, `info()`, and `describe()`.

3. Data Cleaning
   Handling missing values and preparing data for analysis.

4. Exploratory Data Analysis (EDA)
   Analyzing relationships between variables such as confirmed cases, deaths, and recoveries.

5. Data Visualization
   Different plots such as bar charts, histograms, and heatmaps are used to visualize trends and correlations.

## Key Insights

* Countries with higher confirmed cases generally show higher recovery numbers.
* Death rates vary significantly across regions.
* Certain WHO regions have higher growth in weekly confirmed cases.

## Conclusion

This project demonstrates how Python can be used to perform exploratory data analysis on real-world datasets. The analysis helps in understanding the spread and impact of COVID-19 across different regions.
