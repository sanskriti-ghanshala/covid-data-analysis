
# COVID-19 Data Analysis


## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a global COVID-19 dataset using Python.  
The purpose of this analysis is to understand the spread and impact of COVID-19 across different countries by analyzing confirmed cases, deaths, recoveries, and active cases.

Data visualization techniques are used to identify patterns and trends in the pandemic data.


## 🎯 Objectives
The main objectives of this project are:

- Analyze global COVID-19 data
- Identify countries with the highest number of **confirmed cases**
- Compare **death and recovery rates**
- Visualize the relationship between different COVID-19 statistics
- Present insights using graphs and charts



## 📊 Dataset Description

The dataset contains COVID-19 statistics for different countries with the following columns:

| Column Name | Description |
|-------------|-------------|
| Country/Region | Name of the country or region |
| Confirmed | Total confirmed COVID-19 cases |
| Deaths | Total number of deaths |
| Recovered | Total recovered cases |
| Active | Active COVID-19 cases |
| New cases | New confirmed cases |
| New deaths | New deaths reported |
| New recovered | Newly recovered cases |
| Deaths / 100 Cases | Death rate per 100 cases |
| Recovered / 100 Cases | Recovery rate per 100 cases |
| Deaths / 100 Recovered | Death ratio among recovered cases |
| Confirmed last week | Cases recorded in the previous week |
| 1 week change | Change in confirmed cases within a week |
| 1 week % increase | Percentage increase in cases |
| WHO Region | WHO region classification |



## 🛠️ Technologies Used

The project was implemented using the following tools and libraries:

- **Python**
- **Pandas** – Data loading and data manipulation
- **NumPy** – Numerical operations
- **Matplotlib** – Data visualization
- **Seaborn** – Statistical data visualization
- **Jupyter Notebook**



## 📈 Analysis Performed

### 1️⃣ Top Countries with Highest Confirmed Cases
A bar chart was created to identify the countries with the highest number of confirmed COVID-19 cases.

### 2️⃣ Death vs Recovery Comparison
The relationship between deaths and recovered cases was analyzed using visualization.

### 3️⃣ Active Cases Distribution
Analysis of active cases across different countries was performed to understand the spread of the virus.

### 4️⃣ Correlation Analysis
A heatmap was created to examine correlations between numerical features such as confirmed cases, deaths, recoveries, and active cases.



## 🔍 Key Insights

- Some countries recorded significantly higher confirmed cases compared to others.
- The number of recoveries increased alongside confirmed cases in many regions.
- Death rates varied across different countries.
- Data visualization helped identify relationships between COVID-19 statistics.



## ✅ Conclusion
This project demonstrates how **data analysis and visualization techniques** can be applied to real-world pandemic datasets. By analyzing COVID-19 data, we can better understand global trends, case distribution, and the overall impact of the pandemic.
